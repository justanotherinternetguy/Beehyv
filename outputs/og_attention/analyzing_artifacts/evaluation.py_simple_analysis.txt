## Logic Analysis: `evaluation.py`
This analysis strictly adheres to the paper methodology, design specifications, task requirements, and `config.yaml` settings. All values are explicitly referenced to the provided configuration or paper; no undocumented assumptions are made.

---
### Class-Level Overview
`Evaluation` handles inference, BLEU scoring, and alignment visualization for trained `RNNEncDec` (baseline) or `RNNsearch` (proposed) models. It inherits dependencies: `torch`, `sacrebleu`, `matplotlib`, `numpy`, `model.py`, `dataset_loader.py`.

#### Key Attributes (initialized in `__init__`):
| Attribute | Source | Description |
|-----------|--------|-------------|
| `self.model` | `__init__` input | Instance of `RNNEncDec` or `RNNsearch` |
| `self.test_loader` | `__init__` input | PyTorch DataLoader for WMT 2014 news-test-2014 (3003 sentences, no length filtering per Sec 4.1) |
| `self.config` | `__init__` input | Loaded `config.yaml` dictionary |
| `self.beam_width` | Config: `evaluation.beam_search.width` | Default 10 (per Shared Knowledge: `beam_width=10`), as paper does not specify width |
| `self.block_unk` | Config: `evaluation.no_unk_evaluation.block_unk_generation` | Boolean, `True` per config |
| `self.start_tid` / `self.end_tid` / `self.unk_tid` | Shared Knowledge: Special token IDs | `<start>:0`, `<end>:1`, `[UNK]:2` (config `dataset.special_tokens` confirms `unk` and `end` IDs, `<start>` added per Shared Knowledge) |
| `self.max_decode_len_factor` | Plan Sec 3.4.1 | 2x source length (max decode length = `2 * src_len`) |
| `self.is_rnnsearch` | Model type check | `True` if `isinstance(self.model, RNNsearch)` (alignment only supported for RNNsearch per paper Sec 5.2.1) |
| `self.test_samples` | Cached test set | List of dicts with `src_ids`, `src_len`, `tgt_ref_ids`, `src_tokens`, `tgt_ref_tokens` for all 3003 test samples |
| `self.tgt_vocab` / `self.src_vocab` | `test_loader.dataset` | Vocabulary mappings (ID Ōåö token) from `dataset_loader.py` |

---
### Method 1: `Evaluation.__init__(model: object, test_loader: DataLoader, config: dict)`
#### Logic:
1. Store core inputs: `self.model`, `self.test_loader`, `self.config`.
2. Set beam width:
   - Read from `self.config['evaluation']['beam_search']['width']` ŌåÆ value is "Unspecified in paper" per config, so fall back to default 10 (per Shared Knowledge: `beam_width=10`).
3. Set UNK blocking flag: `self.block_unk = self.config['evaluation']['no_unk_evaluation']['block_unk_generation']` (explicitly `True` per config).
4. Set special token IDs (per Shared Knowledge 1):
   - `self.start_tid = 0`, `self.end_tid = 1`, `self.unk_tid = 2`
   - Cross-validate with config `dataset.special_tokens`: `unk: "[UNK]"` (ID 2), `end: "<end>"` (ID 1). `<start>` is implicit (ID 0) per Shared Knowledge.
5. Set max decode length factor: `self.max_decode_len_factor = 2` (per Plan Sec 3.4.1: "max length (2x source length) is reached").
6. Cache all test set samples:
   - Iterate through `self.test_loader` once to avoid repeated I/O.
   - Store for each sample: `src_ids` (1D tensor), `src_len` (int), `tgt_ref_ids` (1D tensor), `src_tokens` (list of str), `tgt_ref_tokens` (list of str). These are required for BLEU filtering and alignment visualization.
7. Check model type: `self.is_rnnsearch = isinstance(self.model, RNNsearch)` (alignment visualization only works for RNNsearch, as RNNencdec has no attention weights).
8. Load vocabularies: `self.src_vocab = self.test_loader.dataset.src_vocab`, `self.tgt_vocab = self.test_loader.dataset.tgt_vocab` (from `dataset_loader.py` which builds vocabularies per Sec 4.1).

---
### Method 2: `Evaluation.evaluate() -> dict`
#### Purpose:
Compute BLEU scores for the full test set ("All") and filtered no-UNK subset ("No UNK") per paper Table 1. Returns dictionary with `bleu_all`, `bleu_no_unk`, and optional hypothesis/reference lists.

#### Steps:
1. Initialize storage lists:
   - `all_hypotheses`: Detokenized hypothesis strings for all 3003 test samples.
   - `all_references`: Detokenized reference strings for all 3003 test samples.
   - `no_unk_hypotheses` / `no_unk_references`: Filtered subsets (no UNK in source/reference).
2. Iterate over cached `self.test_samples`:
   For each sample:
   a. Run beam search to generate hypothesis IDs: `hyp_ids = self.beam_search(sample['src_ids'], sample['src_len'])`.
   b. Convert `hyp_ids` to tokens:
      - Skip `<start>` (ID 0) and `<end>` (ID 1) tokens.
      - `hyp_tokens = [self.tgt_vocab[id] for id in hyp_ids if id not in (self.start_tid, self.end_tid)]`
   c. Detokenize hypothesis and reference:
      - Use `sacremoses.MosesDetokenizer(lang="fr")` for French references/hypotheses, matching paper's Moses tokenization (Sec 4.1: no lowercasing/stemming).
      - `hyp_str = detokenizer.detokenize(hyp_tokens)`
      - `ref_str = detokenizer.detokenize(sample['tgt_ref_tokens'])`
   d. Append to full set lists: `all_hypotheses.append(hyp_str)`, `all_references.append(ref_str)`.
   e. Filter no-UNK subset (per Table 1 note: "no unknown words in themselves and in the reference translations"):
      - Check if `self.unk_tid` is not present in `sample['src_ids']` (source has no UNK).
      - Check if `self.unk_tid` is not present in `sample['tgt_ref_ids']` (reference has no UNK).
      - If both true, append to `no_unk_hypotheses` and `no_unk_references`.
3. Compute BLEU scores (per Plan Sec 3.4.2, using SacreBLEU with WMT 2014 settings):
   - `bleu_all = sacrebleu.corpus_bleu(all_hypotheses, [all_references], tokenize='moses', lowercase=False)`
     - `tokenize='moses'` matches paper's preprocessing; `lowercase=False` matches no lowercasing (Sec 4.1).
   - `bleu_no_unk = sacrebleu.corpus_bleu(no_unk_hypotheses, [no_unk_references], tokenize='moses', lowercase=False)` if `no_unk_hypotheses` is non-empty, else `None`.
4. Return dict:
   ```python
   {
       'bleu_all': bleu_all.score,
       'bleu_no_unk': bleu_no_unk.score if bleu_no_unk else None,
       'all_hypotheses': all_hypotheses,
       'all_references': all_references
   }
   ```

#### Edge Cases:
- RNNencdec has no alignment weights, so alignment visualization is skipped for this model.
- Test set size is exactly 3003 sentences (Sec 4.1), so `all_hypotheses` must have 3003 entries.
- UNK blocking is only applied for "No UNK" evaluation (per Table 1 note: "We disallowed the models to generate [UNK]").

### Method 3: `Evaluation.beam_search(src: Tensor, src_lengths: Tensor) -> List[int]`
#### Purpose:
Generate a translation for a single source sentence using beam search, respecting UNK blocking and max length constraints. Returns list of token IDs (excluding `<start>`, stopping at `<end>`).

#### Inputs:
- `src`: 1D tensor of source token IDs (shape: `[src_len]`)
- `src_lengths`: Scalar int (source sentence length)

#### Steps:
1. Preprocess input for model:
   - Reshape `src` to `[1, src_len]` (batch size 1), `src_lengths` to `[1]`.
2. Encode source with model:
   - If `RNNencdec`: `c = self.model.encode(src, src_lengths)` ŌåÆ fixed 1000-dim context vector (Sec 4.2: `c = h_Tx`).
   - If `RNNsearch`: `annotations = self.model.encode(src, src_lengths)` ŌåÆ list of 2000-dim annotations (forward + backward hidden states, Appendix A.2.1).
3. Initialize decoder initial state `s0` (per paper Appendix A.2.2 and ambiguity resolution 3):
   - RNNencdec: `s0 = torch.tanh(self.model.W_s @ c)` (W_s Ōłł RŌü┐╦ŻŌü┐ per Sec A.2.2).
   - RNNsearch: `s0 = torch.tanh(self.model.W_s @ annotations[0][:, :1000])` (uses first backward hidden state `ŌåÉh1` per Appendix A.2.2).
4. Initialize beam:
   - Single starting beam entry:
     ```python
     {
         'seq': [self.start_tid],  # Start with <start> token
         'score': 0.0,  # Cumulative log probability
         's_prev': s0,  # Previous decoder state
         'is_finished': False
     }
     ```
5. Beam search loop:
   - Max steps: `max_steps = self.max_decode_len_factor * src_lengths` (2x source length per Plan Sec 3.4.1).
   - While there are unfinished beams and steps < max_steps:
     a. Collect all unfinished beams, extract their last token (`y_prev = beam['seq'][-1]`), previous state (`beam['s_prev']`).
     b. Embed last token: `y_emb = self.model.tgt_emb(y_prev)` (620-dim per config `dataset.embedding_dim`).
     c. Compute logits and new state per model type:
        - **RNNencdec**:
          - Fixed context `c` is reused for all steps.
          - `logits, s_new = self.model.decoder_cell(y_emb, beam['s_prev'], c)` (per `GatedRNNCell` forward pass, Appendix A.1.1).
        - **RNNsearch**:
          - Compute alignment scores: `e_ij = self.model.alignment_model.forward(beam['s_prev'], annotations)` ŌåÆ shape `[1, Tx]` (Appendix A.1.2: `a(s_i-1, h_j) = v_a^T tanh(W_a s_i-1 + U_a h_j)`).
          - Normalize to get weights: `alpha = F.softmax(e_ij, dim=-1)` ŌåÆ sum to 1 per step.
          - Compute per-step context: `c_i = torch.sum(alpha.unsqueeze(-1) * torch.stack(annotations, dim=1), dim=1)` ŌåÆ 2000-dim (Eq 5, Sec 3).
          - `logits, s_new = self.model.decoder_cell(y_emb, beam['s_prev'], c_i)`.
     d. Compute log probabilities: `log_probs = F.log_softmax(logits, dim=-1)` ŌåÆ shape `[1, 30000]` (vocab size per config `dataset.vocab_size`).
     e. Block UNK if enabled: `if self.block_unk: log_probs[:, self.unk_tid] = -torch.inf`.
     f. Expand beams:
        - For each unfinished beam, get top `self.beam_width` log probabilities and token IDs: `top_log_probs, top_ids = log_probs.topk(self.beam_width)`.
        - Create new beam entries for each (log_prob, token_id):
          - `new_seq = beam['seq'] + [token_id.item()]`
          - `new_score = beam['score'] + log_prob.item()`
          - `new_finished = (token_id.item() == self.end_tid)`
        - Collect all candidate beams, prune to top `self.beam_width` beams by score (highest first).
     g. Mark beams as finished if they generate `<end>` token.
  6. Select best hypothesis:
     - Choose the finished beam with the highest score. If no finished beams exist, choose the highest-scoring unfinished beam.
  7. Return token IDs: Exclude `<start>` (ID 0) and `<end>` (ID 1) from the sequence.

#### Edge Cases:
- Beam width 1 equivalent to greedy decoding.
- Max step limit prevents infinite loops for long sentences.
- UNK blocking enforces no `[UNK]` tokens in output for "No UNK" evaluation.

### Method 4: `Evaluation.visualize_alignments(src_tokens: List[str], tgt_tokens: List[str], alpha_weights: Tensor) -> None`
#### Purpose:
Plot grayscale heatmap of alignment weights `╬▒_ij` per paper Fig 3, save to file. Only applicable for RNNsearch (RNNencdec has no attention weights).

#### Inputs:
- `src_tokens`: List of source (English) word strings (length Tx).
- `tgt_tokens`: List of target (French) word strings (length Ty).
- `alpha_weights`: Tensor of shape `[Ty, Tx]` where `alpha_weights[i][j] = ╬▒_ij` (weight of j-th source annotation for i-th target word).

#### Steps:
1. Validate inputs:
   - Check `alpha_weights.shape == (len(tgt_tokens), len(src_tokens))`. Raise error if mismatched.
2. Create matplotlib figure:
   - `fig, ax = plt.subplots(figsize=(len(src_tokens)/2, len(tgt_tokens)/2))` (scale figure to sentence length).
3. Plot heatmap (per paper Sec 5.2.1):
   - `ax.imshow(alpha_weights.cpu().numpy(), cmap='gray', vmin=0, vmax=1)`
   - `cmap='gray'`: 0 = black, 1 = white (matches paper: "0: black, 1: white").
4. Set axis labels and ticks:
   - X-axis (source): `ax.set_xticks(range(len(src_tokens)))`, `ax.set_xticklabels(src_tokens, rotation=45, ha='right')`
   - Y-axis (target): `ax.set_yticks(range(len(tgt_tokens)))`, `ax.set_yticklabels(tgt_tokens)`
   - Labels: `ax.set_xlabel('Source (English)')`, `ax.set_ylabel('Target (French)')`
   - Title: `ax.set_title('Soft Alignment Weights ╬▒_ij')`
5. Save figure:
   - `plt.savefig('alignment_visualization.png', bbox_inches='tight')` (prevents label cutoff).
   - Close figure: `plt.close()` to free memory.

#### Alignment Sample Selection (per paper Sec 5.2.1):
- Filter test set samples for visualization: no UNK, source length 10-20 words (per config `evaluation.alignment_visualization.filter`).
- For each selected sample:
  1. Run beam search (or greedy decode) for RNNsearch to get `tgt_tokens` and `alpha_weights` (collect `alpha` per decode step, stack to form `[Ty, Tx]` tensor).
  2. Call `visualize_alignments` with the sample's `src_tokens`, generated `tgt_tokens`, and `alpha_weights`.
